Bring Twitter to your WordPress site with powerfull widget uses the Twitter API version 1.1.

== Description ==
This plugin comes with lots of visual options and easy to customize. Uses the powerful Twitter api such as 
mentions timeline, user timeline, and home timeline. You can built your own twitter 
template(s) with just simple steps, customize the stylesheet, and even share the template to other peoples. 

<h3><strong>Key Features & Options</strong></h3>
The powerfull of <a href="https://dev.twitter.com/docs/api">Twitter REST API v.1.1</a><br />
Show user, home or mentions timeline<br />
Interactive dialog interface.<br />
Tweet actions favorites, reply, retweet and follow button<br />
Multi widget.<br />
Optional avatar height and width.<br />
Number of tweets to display.<br />
Tweet display style, fader, ticker or default.<br />
Text before and after your tweet.<br />
Template tags for creating your new template<br />
Suitable for developper<br />
And more...<br />

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
<ul>
<li>Go to the menu 'Plugins' -> 'Install' and search for 'Jet Tweet'</li>
<li>Click 'install'</li>
</ul>
